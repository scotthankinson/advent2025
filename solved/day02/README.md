# Day 2 - Working Directory

## Quick Start

1. Paste example input into `example.txt`
2. Paste actual puzzle input into `input.txt`
3. Code solution in `solution.ts`
4. Test with example: `npx ts-node src/solution.ts example.txt`
5. Run on actual: `npx ts-node src/solution.ts`

## Files

- `input.txt` - Your actual puzzle input from Advent of Code
- `example.txt` - Example input from puzzle description (for testing)
- `solution.ts` - Your solution code

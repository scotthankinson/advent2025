# Working Directory

## Quick Start

1. Paste example input into `example.txt`
2. Paste actual puzzle input into `input.txt`
3. Paste puzzle descriptions into `pt1.txt` and `pt2.txt`
4. Code solution in `solution.ts`
5. Test with example: `npx ts-node src/solution.ts example.txt`
6. Run on actual: `npx ts-node src/solution.ts`
7. Update `test.ts` with expected values and run: `npx ts-node src/test.ts`

## Files

- `input.txt` - Your actual puzzle input from Advent of Code
- `example.txt` - Example input from puzzle description (for testing)
- `pt1.txt` - Part 1 puzzle description
- `pt2.txt` - Part 2 puzzle description
- `solution.ts` - Your solution code
- `test.ts` - Automated test harness
